// HERE IS HOW TO BUILD A VERSION

1.  Check in all code.

2.  Generate a translation pot template here:
http://wordpress.org/extend/plugins/advertising-manager/admin/

3.  Copy the translation template to the pootle server:
mv advertising-manager.pot advman.pot
scp advman.pot pootle@openx.org:/home/pootle/Pootle/po/advman/templates/
rm advman.pot

4.  Update the templates:
http://translate.openx.org/projects/advman/admin.html?doupdatelanguage=1&updatelanguage=en

4.  Translate (at least English!):
http://translate.openx.org/en/advman/index.html?editing=1

5.  Generate the .mo file for all languages:
http://translate.openx.org/en/advman/advman-en_US.mo
http://translate.openx.org/es/advman/advman-es_ES.mo

6.  Copy all .mo files to the languages directory

7.  Increase the ADVMAN_VERSION at the top of advertising-manager.php

8.  Increate the version in the comments at the top of advertising-manager.php

9.  Update the Readme.txt file with any changes in documentation.

10.  Follow the build process below:

mkdir build
cd build
svn export http://svn.wp-plugins.org/advertising-manager/trunk
mv trunk advertising-manager
zip -r advertising-manager.zip advertising-manager/
